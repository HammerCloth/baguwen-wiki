# 1161.如何显示前 50 行？

> 原文：[https://zwmst.com/5623.html](https://zwmst.com/5623.html)

   [ *MySQL* ](https://zwmst.com/mysql)*[ <time datetime="2021-10-28T00:32:06+08:00"> 2021-10-27 </time> ](https://zwmst.com/5623.html)  在 Mysql 中，使用以下代码查询显示前 50 行：

```
SELECT*FROM
LIMIT 0,50;
```*
