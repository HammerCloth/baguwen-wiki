# vim命令

> 原文：[https://zwmst.com/600.html](https://zwmst.com/600.html)

   [ *Linux* ](https://zwmst.com/linux)*[ <time datetime="2021-08-14T07:38:08+08:00"> 2021-08-13 </time> ](https://zwmst.com/600.html)  打开文件并跳到第 10 行：vim +10 filename.txt 。
