   [ *数据库* ](https://zwmst.com/%e6%95%b0%e6%8d%ae%e5%ba%93)*[ <time datetime="2021-09-25T02:10:51+08:00"> 2021-09-24 </time> ](https://zwmst.com/4009.html)  第三范式定义是，满足第二范式，并且表中的列不存在对非主键列的传递依赖。除了主键订单编号外，顾客姓名依赖于非主键顾客编号。*
