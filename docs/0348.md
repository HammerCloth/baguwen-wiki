# 什么是聚簇索引？

> 原文：[https://zwmst.com/661.html](https://zwmst.com/661.html)

   [ *MySQL* ](https://zwmst.com/mysql)*[ <time datetime="2021-08-14T07:52:05+08:00"> 2021-08-13 </time> ](https://zwmst.com/661.html)  聚簇索引：将数据存储与索引放到了一块，找到索引也就找到了数据
